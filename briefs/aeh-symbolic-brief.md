Pre-checked with fresh throwaway code (main session, 2026-07-23; script deliberately not provided):

- **The cylinder measure is a Bernoulli product measure — exactly.** Theorem 14.15.1.5 gives each length-`n` word one odd class mod `2^(S+1)`, whose Haar-mass among odd `2`-adics is `2^(−S) = Π_i 2^(−(m_i+r_i))`. This factorizes: letters i.i.d. with `P(m, r) = 2^(−(m+r))`, i.e. **`m` and `r` independent geometric(1/2)**. Exact count verified for all `(m,r)` with `m, r ≤ 5` (one class mod `2^(m+r+1)` each) and all `81` two-letter words with letters `≤ 3` (one class mod `2^(S+1)` each), 0 failures.
- **The coding intertwines `G` with the full shift and pushes Haar to Bernoulli:** `G(y) mod 256` over `200,000` uniform odd `y` (`< 2^40`) is uniform over odd residues (worst cell deviation `0.085` relative, consistent with sampling noise at `~3σ` for the worst of 128 cells). With 14.15.3.1–.2's nested-cylinder machinery this upgrades to: `(odd Z₂ minus a Haar-null exceptional set, Haar, G)` is **measure-isomorphic to the Bernoulli shift** on alphabet `{(m,r)}` with weights `2^(−(m+r))`.

**The two known hard points (main-session assessment — engage them, don't discover them):**
- *Time change.* Window states (aeh 13.2) are indexed per `F`-block; letters are indexed per `G`-step; a `G`-step is `m` `T`-steps and the `F`-block/`T`-step correspondence runs through the seam (14.14.7). Empirical-frequency equivalence across these time scales needs the induced-map/return-time argument done explicitly (expected letter length is finite and explicit: `E[m] = 2` under the Bernoulli law), not waved at.
- *The depth marginal.* `π_k` includes the capped depth `min(d, D_k)` with its stationary window-chain law; the symbolic side carries no explicit depth. The route is the exact per-step laws (`d_+ = m_+ + a_+`, stage3): show the capped depth is determined by bounded letter history up to an exceptional set of vanishing frequency, or scope the theorem to the component where the dictionary is exact and record the depth component's status precisely.
