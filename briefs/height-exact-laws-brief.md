## Provenance and pre-check

A frontier seed handed forward from the previous session observed, from the two published tables alone, that `H_{n,n}` grows at the per-window CRT-modulus rate (ratios → `72 = 2³·3²` on `((2,1))^∞`, → `12 = 2²·3¹` on `(1,1)^∞` negative-sector). The main session pre-checked this 2026-07-17 with fresh throwaway code and found it is **exact, not asymptotic**; author authorized recording it 2026-07-17. Pre-check facts (script deliberately not provided — write fresh code):

- **Positive sector, `((2,1))^∞`:** `H_{n,n} = 4·72ⁿ − 5` exactly, verified `n = 1..12` (matches all 8 published values); brute-force minimality confirmed at `n = 1, 2` without any class reasoning.
- **Negative sector, `(1,1)^∞`:** `H⁻_{n,n} = 2·12ⁿ − 1` exactly, verified `n = 1..10` (matches all 5 published values); brute force at `n = 1, 2`.
- **Mechanism, fully identified:** (i) both class facts are iff's — the forward cylinder class (14.15.1.4–.5) and the backward admissibility class (14.15.5.1 / signed 14.15.6.4) — so `R^σ_{n,n}` for a constant word `((m,r))^∞` is exactly one residue class mod `Q_n = 2^{(m+r)n+1}·3^{mn}` intersected with the sign condition, liveness of `y₀`, and liveness of the deepest chain door; (ii) the class representative is the word's own fixed point `y* = β/(1−α)` (the constant chain at `y*` satisfies every window — one line via 14.15.4.1's formula); (iii) candidates are `y₀ = y* + kQ_n` (`k` signed by sector), `y₀ ≡ y* (mod 3)` since `3 | Q_n`, so `y₀` is live automatically whenever `y*` is; (iv) the deepest chain door is `y_{−n} = y* + (y₀ − y*)/A_n = y* + k·Q_n/A_n`, and `Q_n/A_n = 2^{2(m+r)n+1} ≡ 2 (mod 3)` for **every** constant letter and every `n` — so deepest-door liveness reads `y* + 2k ≢ 0 (mod 3)`, an `n`-independent condition on `k mod 3`; (v) the minimal viable `k` is therefore a constant: `k = 2` for `((2,1))^∞` positive (`y* = −5 ≡ 1`, dead iff `k ≡ 1 mod 3`, and `k = 1` is indeed dead), `k = 1` for `(1,1)^∞` negative (`y* = 1`, dead iff `k ≡ 2 mod 3`). Hence the closed forms, with the empirical dead/live pattern over `k = 1..9` matching the mod-3 law exactly in both instances.

## The framing mandate

The two published tables were left as "single named instance, not a growth study"; these theorems close them — for periodic words whose fixed point is a known live integer, the height's growth is an elementary exact law, trivial in *both* directions (the upper bound because the fixed point caps the class rep, the lower bound because the class rep *is* the fixed point).
