## Setup and proof route (main-session guidance; own the proofs)

Fix a period `P = ((m_0,r_0),…,(m_{p−1},r_{p−1}))`, `W = P^∞`, `S_P, M_P, A_P, B_P` (deepest-first, 14.15.5.1's order note), `y* = B_P/(1−A_P) = a/q` lowest terms, `Q_n = 2^{nS_P+1}·3^{nM_P}`, `g_P = 2^{S_P}·3^{M_P} mod q`.

1. **Adelic anchor lemma.** (i) `q | 2^{S_P} − 3^{M_P}`, hence `gcd(q,6) = 1` and `y* ∈ Z₂ ∩ Z₃`. (ii) `y₂(W) = y₃(W) = y*` as elements of `Z₂` resp. `Z₃`: the `3`-adic half is `A_n y* + B_n = y*` with `v₃(A_P^n) = nM_P` (the 14.15.8.1(3) one-liner, composed); for the `2`-adic half, check what `14.15.3`'s periodic-word identification already provides and cite it — if the forward statement (`y₂` of a periodic word is the affine fixed point) is not already on file at citation grade, prove it: the follower classes mod `2^{S_n+1}` are nested and nonempty (14.15.1.5), and the composed affine identity plus the contraction argument of `14.15.8.2`'s forward half (run at period level: an odd-denominator rational within `2^{−(S_P+1)}`-distance of `y*` satisfies all `p` stratum congruences of one period, since each rotated fixed point does — see (iii)) pins the limit. (iii) **Rotation lemma**: the rotated fixed points `y*_0 = y*, y*_{i+1} = α_i y*_i + β_i` satisfy `v₂(y*_i + 1) = m_i` and `v₂(3^{m_i}q^*_i − 1) = r_i` (`q^*_i := (y*_i+1)/2^{m_i}`) — the per-position stratum facts, `2`-adically — and **all `p` rotations share the same reduced denominator `q`** (prove both directions: the affine step multiplies the denominator by at most a power of `2`, which cannot enter a reduced odd denominator, and the step is invertible over `Z[1/6]`). The probe's M4 verified all of this on 5 words compute-don't-assume; now it is to be proved for every `P`.
2. **Class theorem at whole-period windows.** Verbatim 14.15.8.2 with the composed objects: the odd integers following `W`'s length-`np` forward prefix and admitting the chain to depth `np` are exactly `{y ≡ ρ_n (mod Q_n)}`, `ρ_n = a·q^{−1} mod Q_n`; `R^σ_{np,np}` is that class ∩ sign ∩ deepest-door liveness (`y₀`-liveness automatic when `3 ∤ a`; handle the `3 | a` question — can it occur? `a ≡ ±... ` — settle it, one line, rather than assuming).
3. **The identities, unified.** With the class parametrized as in 14.15.8.3: `j_n = −a·2^{−1}·g_P^{−n} mod q` purely periodic at exactly `ord_q(g_P)`; deepest door `y* + (j_n + σkq)·2^{2nS_P+1}/q` with `2^{2nS_P+1} ≡ 2 (mod 3)`; the death law and first-viable rule via `t_n = (a + 2j_n)q^{−1} mod 3`; closed forms `H⁺ = ρ_n + [t_n=0]Q_n`, `H⁻ = (1+[t_n=2])Q_n − ρ_n`. **Integer case folded in**: when `q = 1` and `2|y*| < Q_1`, recover 14.15.7.5's capped/escaping dichotomy as the degenerate case (this discharges 14.15.7.5's H1 — item 1(ii) makes "the orbit realizes `W`" a theorem, not a hypothesis — and leaves H2 as the one surviving smallness condition for the capped-sector equality; state the reconciliation precisely, cross-referencing, never editing, 14.15.7).
4. **Corollaries, with the probe's corrections built in.** (P1) `v_n` purely periodic from `n = 1`, period exactly `ord_q(g_P)`, both sectors. (P2) `v_n ≥ 1/q`; **sharp constant = `min` over the orbit `O = −a·2^{−1}⟨g_P⟩` of `V_σ(j)` where `V_+(j) = j/q + [t(j)=0]` and `V_−(j) = 1 + [t(j)=2] − j/q`** — the probe's corrected formula; state when it reduces to `j_min/q` and exhibit the failure instances from the probe as citations.
