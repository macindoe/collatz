## 14.15. The itinerary language: the cylinder theorem and the two-sided coding

*(The cylinder theorem below is **not new mathematics about `T`** вҖ” it is the classical Collatz coding fact (parity-vector periodicity, Terras 1976 and Everett 1977; the accelerated form standard, Lagarias's surveys, Wirsching) read in the door/`(m,r)` alphabet fixed by the seam (`14.14`, reverse.md), with its consequences for this program's search space stated once (`14.15.2`) and its two-sided `Z_2 Г— Z_3` extension formulated, with no new leverage on the Bridge (`14.15.3`).)*

### 14.15.1. The finite-itinerary cylinder theorem

**Definition 14.15.1.1 (stratum, general).** For odd `y > 0`, `m(y) := v_2(y+1)`, `q(y) := (y+1)/2^{m(y)}`, `r(y) := v_2(3^{m(y)} q(y) - 1)`, `stratum(y) := (m(y), r(y))` вҖ” the same quantities `14.14.4` (reverse.md) names `(m,r)`, defined here for *every* odd `y`, not only live doors: nothing in the definitions of `m`, `q`, `r` uses `3 вҲӨ y`.

**Remark (well-defined beyond live doors, already on file).** `14.14.3.1`'s formula for `G` uses exactly `m`, `q`, `r` above and likewise needs no coprimality hypothesis; `14.14.7.1` already proves `G(y) = T^{m(y)}(y)` and that `T` is total and never outputs a multiple of `3`, for *every* odd `x`, live door or not (its own cross-check paragraph makes this explicit). So `stratum` and `G` are total on all positive odd integers, and `G`'s output is always a live door regardless of its input's relation to `3`. This section uses that generality throughout: words below are not restricted to sequences of live doors, only their first letter's realizability as an actual chain is (Corollary `14.15.1.7`).

**Definition 14.15.1.2 (word, follower).** A **word** of length `n` is `W = ((m_0,r_0), вҖҰ, (m_{n-1},r_{n-1}))` with every `m_i, r_i вүҘ 1`. An odd integer `y` **follows** `W` if, writing `y_0 := y`, `y_{i+1} := G(y_i)`, `stratum(y_i) = (m_i,r_i)` for every `i = 0, вҖҰ, n-1`. Write `S(W) := ОЈ_i (m_i+r_i)`.

**Lemma 14.15.1.3 (single-stratum cylinder and level shift).** Fix `m, r вүҘ 1`.

```text
(i)  stratum(y) = (m,r)  iff  y вүЎ 2^m - 1 (mod 2^(m+1))  and, writing q=(y+1)/2^m,
                               q вүЎ 3^(-m)(1+2^r) (mod 2^(r+1))
     -- equivalently, iff y is congruent to a single fixed residue mod 2^(m+r+1).
(ii) for y with stratum(y) = (m,r), and any integer t,
     G(y + 2^(m+r+1) t) = G(y) + 2В·3^m t                                exactly.
```

**Proof.** (i) `v_2(y+1) = m` iff `y+1 = 2^mВ·(odd)`, i.e. `y вүЎ 2^m-1 (mod 2^{m+1})`; this pins `y+1 mod 2^{m+r+1}`, hence `q=(y+1)/2^m mod 2^{r+1}` exactly. `v_2(3^mq-1)=r` iff the coefficient of `2^r` in `3^mq-1` is odd, i.e. `3^mq вүЎ 1+2^r (mod 2^{r+1})`, i.e. `q вүЎ 3^{-m}(1+2^r) (mod 2^{r+1})` (`3^m` a unit mod `2^{r+1}`). Both conditions together pin `y` to a single residue mod `2^{m+r+1}`.

(ii) Let `y' = y+2^{m+r+1}t`. Since `m+r+1 > m`, `y' вүЎ y (mod 2^{m+1})`, so `m(y')=m`, and `q' := (y'+1)/2^m = q+2^{r+1}t`. Write `3^mq-1 = 2^rc` with `c` odd (`v_2=r` by hypothesis); then `3^mq'-1 = 2^rc + 3^m2^{r+1}t = 2^r(c+2В·3^mt)`, and `c+2В·3^mt` is odd, so `v_2(3^mq'-1)=r` too (consistent with (i)) and `G(y') = (3^mq'-1)/2^r = c+2В·3^mt = G(y)+2В·3^mt`. вҲҺ

**Lemma 14.15.1.4 (composed cylinder and level shift).** For a word `W` of length `n`, `S=S(W)`, `M := ОЈ_i m_i`: the followers of `W` form exactly one residue class `y_W` mod `2^{S+1}`, and for every integer `t`,

```text
G^n(y_W + 2^(S+1) t) = G^n(y_W) + 2В·3^M t                                exactly.
```

**Proof.** Induction on `n`. `n=1` is Lemma `14.15.1.3` (`S=m_0+r_0`, `M=m_0`). Suppose the claim holds for the length-`n` prefix `W_n`, with representative `y^{(n)}` and `G^n(y^{(n)}+2^{S_n+1}t) = z + 2В·3^{M_n}t` (`z := G^n(y^{(n)})`, odd, since `G` always outputs an odd integer). Append `(m_n,r_n)`: `y` follows `W_{n+1}` iff `y=y^{(n)}+2^{S_n+1}t` (some `t`, by hypothesis) and `stratum(z+2В·3^{M_n}t) = (m_n,r_n)`. By Lemma `14.15.1.3`(i), the latter holds iff `z+2В·3^{M_n}t вүЎ w_{m_n,r_n} (mod 2^{m_n+r_n+1})` for the fixed residue `w_{m_n,r_n}` of stratum `(m_n,r_n)`; since `z` and `w_{m_n,r_n}` are both odd, dividing the (even) difference by `2` and using that `3^{M_n}` is a unit mod every power of `2` gives a unique solution `t вүЎ t_0 (mod 2^{m_n+r_n})`. So `t=t_0+2^{m_n+r_n}u`, and `y = y^{(n)}+2^{S_n+1}t_0 + 2^{S_n+1+m_n+r_n}u =: y^{(n+1)} + 2^{S_{n+1}+1}u` (`S_{n+1}=S_n+m_n+r_n`) вҖ” the single-class claim for `W_{n+1}`. For the shift identity, `w_1' := z+2В·3^{M_n}t_0 = G^n(y^{(n+1)})` lies on stratum `(m_n,r_n)` by construction, and `z+2В·3^{M_n}t = w_1' + 2В·3^{M_n}(t-t_0) = w_1' + 2^{m_n+r_n+1}В·(3^{M_n}u)` (substituting `t=t_0+2^{m_n+r_n}u`), so Lemma `14.15.1.3`(ii) applies at `w_1'` with integer shift-count `t'' = 3^{M_n}u`: `G(w_1'+2^{m_n+r_n+1}t'') = G(w_1')+2В·3^{m_n}t'' = G^{n+1}(y^{(n+1)}) + 2В·3^{M_n+m_n}u`, giving `G^{n+1}(y) = G^{n+1}(y^{(n+1)}) + 2В·3^{M_{n+1}}u` (`M_{n+1}=M_n+m_n`), completing the induction. вҲҺ

**Theorem 14.15.1.5 (the finite-itinerary cylinder theorem).** For every word `W = ((m_0,r_0),вҖҰ,(m_{n-1},r_{n-1}))`, `S = ОЈ_i(m_i+r_i)`:

```text
The odd integers following W (y_i on stratum (m_i,r_i) for all i, y_{i+1} = G(y_i))
form exactly one odd residue class mod 2^(S+1).
```

**Proof.** The single-class clause of Lemma `14.15.1.4`, applied to the full word (`n` = word length). вҲҺ

**Corollary 14.15.1.6 (completeness).** Every word over the alphabet `{(m,r) : m,r вүҘ 1}` is realized: the residue class of Theorem `14.15.1.5` is nonempty (a residue class mod any modulus has representatives), and in fact contains infinitely many odd integers.

**Corollary 14.15.1.7 (liveness).** For every realized word `W`, the class of Theorem `14.15.1.5` contains infinitely many *live* doors (`3 вҲӨ y_0`).

**Proof.** The modulus `2^{S+1}` is coprime to `3`; as the class's members increase by `2^{S+1}` each step and `2^{S+1}` is a unit mod `3`, their residues mod `3` cycle through all three values with period `3` in the step index, so at least one in every three consecutive members is coprime to `3`. Only `y_0` needs this check: `y_1,вҖҰ,y_{n-1}` are automatically live regardless (Remark above, `14.14.7.1`). вҲҺ

**Corollary 14.15.1.8 (single-stratum base case).** A single letter `(m,r)` alone is one class mod `2^{m+r+1}` вҖ” Lemma `14.15.1.3`(i) restated as the `n=1` case of Theorem `14.15.1.5`.

**Content вҖ” framing (per the brief's mandate).** This is the classical coding fact for the Collatz map, not new mathematics: Terras (1976) and Everett (1977) proved that the first `n` binary parities of an odd start are periodic in the start value with period `2^n`, each of the `2^n` parity vectors realized on exactly one residue class вҖ” the raw-bit case of Theorem `14.15.1.5`. The accelerated (block-valued) form is standard (Lagarias's surveys and annotated bibliographies; Wirsching's monograph). What this section adds is only: the same fact, read in the door/`(m,r)` alphabet fixed by the seam (`14.14`), tied to the seam's own strata, with its consequences for this program's search space stated once (`14.15.2`). No claim of new dynamics about `T` is made or intended, per the register norm.

An equivalent proof route pulls the cylinders back through the block-map identity `14.14.7.1` letter by letter: each `(m_i,r_i)` expands to the `T`-valuation word `(1,вҖҰ,1,r_i+1)` (`m_i` entries, sum `m_i+r_i`), and `3x+1 = 2^ax'` pulls cylinders back uniquely at each raw step since `3` is a unit mod every power of `2`; the two routes agree because the expansion's sum matches `14.14.4.1`'s own denominator `2^{m+r}`. ### 14.15.3. The two-sided coding

**(a) The future determines the 2-adic coordinate.**

A right-infinite word is `W = ((m_0,r_0),(m_1,r_1),вҖҰ)`, every `m_i,r_i вүҘ 1`. By Theorem `14.15.1.5` applied to its length-`n` prefixes `W_n`, the followers of `W_n` form a nested sequence of residue classes `y^{(n)} mod 2^{S_n+1}` (nesting is immediate from Lemma `14.15.1.4`'s induction: `y^{(n+1)} вүЎ y^{(n)} (mod 2^{S_n+1})`), with `S_n вүҘ 2n вҶ’ вҲһ`. By the nested-cylinder property of the complete metric space `Z_2`, these classes have a unique common point.

**Definition 14.15.3.1.** `y_2(W) вҲҲ Z_2` is this unique point: `y_2(W) вүЎ y^{(n)} (mod 2^{S_n+1})` for every `n`.

**Theorem 14.15.3.2 (injectivity).** Distinct positive odd integers follow distinct right-infinite words.

**Proof.** If `y вү  z` both follow the same `W`, then for every `n` both lie in the class `y^{(n)} mod 2^{S_n+1}` (Theorem `14.15.1.5`), so `y вүЎ z (mod 2^{S_n+1})` for every `n`; since `S_n вҶ’ вҲһ`, this forces `y=z`, contradiction. вҲҺ

**Content.** This is the accelerated form of the classical `2`-adic coding of the Collatz map: the parity-vector periodicity of Terras (1976) and Everett (1977) is the raw-bit case of this same nested-cylinder statement, and the `2`-adic extension of `T` and its explicit conjugacy to a shift-like map (Lagarias 1985; made explicit as the map `ОҰ` by BernsteinвҖ“Lagarias, *The `3x+1` conjugacy map*, Canad. J. Math. 48 (1996), 1154вҖ“1169, `ОҰ вҲҳ S = T вҲҳ ОҰ` for a shift-like `S`) is the identical topology on all of `Z_2`, in the raw parity-bit alphabet rather than the door/`(m,r)` alphabet. Theorem `14.15.3.2` is not a new injectivity fact; it is the standard one, read off Theorem `14.15.1.5` in one line because the seam's coordinates make the cylinders explicit.

**Remark (does `G` extend to `y_2`? вҖ” optional, not fully resolved).** For fixed `W`, `G(y_2(W))` can itself be computed as a `2`-adic integer by passing Lemma `14.15.1.3`(ii)/`14.15.1.4`'s shift formula to the limit along the approximants `y^{(n)}`, realizing the shifted word `(m_1,r_1),(m_2,r_2),вҖҰ` вҖ” so `G` behaves well on the specific sub-Cantor-set of points arising this way. But the *formula* defining `G` routes through `m(y)=v_2(y+1)`, undefined at the single point `y=-1 вҲҲ Z_2` (`v_2(0)=вҲһ`) вҖ” a genuine odd `2`-adic integer, though never itself equal to any `y_2(W)` (every `y_2(W)` has `v_2(y_2(W)+1)=m_0<вҲһ` by construction). Whether some extension of `G` patches this point, or whether other such points exist, is not attempted here; this is recorded as the honest obstruction to a literal continuous self-map of `G` on all of odd `Z_2`, per the brief вҖ” not forced, not resolved.

**(b) The past determines the 3-adic coordinate.**

A left-infinite word is `(вҖҰ,(m_{-2},r_{-2}),(m_{-1},r_{-1}))`, every `m_i,r_i вүҘ 1`. For `n вүҘ 1`, `W_{(-n:0)} := ((m_{-n},r_{-n}),вҖҰ,(m_{-1},r_{-1}))` (a length-`n` word in forward order); let `A_n,B_n` be its composed-affine constants (`14.14.8.2`, reverse.md), `M_n := ОЈ_{i=-n}^{-1} m_i = v_3(A_n)`.

**Theorem 14.15.3.3 (the offsets converge).** `(B_n)_{nвүҘ1}` is `3`-adically Cauchy вҖ” `v_3(B_{n+1}-B_n) = M_n вҶ’ вҲһ` вҖ” hence `B_n вҶ’ y_3 вҲҲ Z_3`, a limit depending only on the left-infinite word.

**Proof.** Passing from `W_{(-n:0)}` to `W_{(-(n+1):0)}` prepends `(m_{-(n+1)},r_{-(n+1)})`; writing `Оұ,ОІ` for that letter's own affine constants (`14.14.4.1`), substituting `G(y_{-(n+1)}) = Оұy_{-(n+1)}+ОІ` into `G^n(u) = A_n u+B_n` gives `A_{n+1}=A_nОұ`, `B_{n+1}=A_nОІ+B_n`. So `B_{n+1}-B_n = A_nОІ`, and `v_3(B_{n+1}-B_n) = v_3(A_n)+v_3(ОІ) = M_n + 0` (`v_3(A_n)=M_n` by `14.14.8.2`; `v_3(ОІ)=0` exactly as in `14.14.4.1`'s own proof, since `m_{-(n+1)} вүҘ 1`). Each `m_i вүҘ 1` gives `M_n вүҘ n вҶ’ вҲһ`, so `(B_n)` is Cauchy in the complete metric space `Z_3` and converges. вҲҺ

**Theorem 14.15.3.4 (the limit is the present door, to every precision).** If a live door `y_0` has an actual integer chain of live-door predecessors `вҖҰ,y_{-2},y_{-1}` with `y_0 = G^n(y_{-n})` realizing `W_{(-n:0)}` for every `n`, then `y_0 = y_3` exactly in `Z_3`.

**Proof.** By Corollary `14.14.8.3` applied to `W_{(-n:0)}`, once `M_n вүҘ k+1`, `y_0 = G^n(y_{-n}) вүЎ B_n (mod 3^{k+1})` вҖ” independent of `y_{-n}`. By Theorem `14.15.3.3`, `v_3(B_n-y_3) вҶ’ вҲһ`. For `n` large enough that both hold at level `k+1`, `y_0 вүЎ y_3 (mod 3^{k+1})`; since this holds for every `k`, `y_0 = y_3` in `Z_3`. вҲҺ

**Content.** Theorem `14.15.3.4` is the limit form of the synchronization corollary `14.14.8.3`, read backward: `14.14.8.3` says a fixed-length itinerary eventually forgets its starting door; this says an infinite backward itinerary forgets it completely and pins the present door's every `3`-adic digit, with no residual dependence on chain depth. Theorem `14.15.3.3`'s Cauchy argument is bookkeeping on top of `14.14.8.2`'s composition law, not a new mechanism: the same offset sequence `14.14.8.3` already uses at each finite precision is, as a sequence, self-convergent.

**(c) The diagonal compatibility locus.**

**Definition 14.15.3.5.** A bi-infinite word `W = (вҖҰ,(m_{-1},r_{-1}),(m_0,r_0),(m_1,r_1),вҖҰ)` yields `(y_2,y_3) вҲҲ Z_2 Г— Z_3` via `14.15.3.1` on its right-infinite half and `14.15.3.3` on its left-infinite half. `W` is **integrally realized** iff some live door `y_0` follows the right-infinite half exactly and has an actual integer chain of live-door predecessors realizing the left-infinite half exactly.

**Remark (abundance, not scarcity, of live continuations).** Every live door has such chains in abundance: `14.1.1` gives infinitely many predecessor choices (one per admissible exit valuation `s`) at every backward step, and `14.5.1` guarantees the `a вүҘ 1` representative doors of any predecessor state are never dead вҖ” so a live continuation exists whenever the chosen predecessor state has depth `вүҘ 2`, which infinitely many of the infinitely many `s`-choices supply. No word is claimed realizable by this remark вҖ” most are not, since `14.15.3.6` below pins any realized word's diagonal point exactly вҖ” only that realizability is never blocked for want of a live door to continue through.

**Theorem 14.15.3.6 (trivial direction).** If `W` is integrally realized by `y_0`, then `y_2 = y_3 = y_0` under the diagonal embeddings `Z вҶӘ Z_2`, `Z вҶӘ Z_3`.

**Proof.** `y_2=y_0`: `y_0` follows every finite prefix of the right-infinite half, so Theorem `14.15.1.5` places it in every class `y^{(n)} mod 2^{S_n+1}`; as `S_n вҶ’ вҲһ`, `y_0=y_2` in `Z_2` (the argument of Theorem `14.15.3.2`, applied to one integer rather than two). `y_3=y_0`: this is exactly Theorem `14.15.3.4`, applied to the actual predecessor chain realizing the left-infinite half. вҲҺ

**Mandatory identifications.**

- **Periodic words.** For a periodic bi-infinite word of period `n`, the forward composed law's fixed point `y^* = B_n/(1-A_n) вҲҲ Z_3` of Corollary `14.14.8.4` *is* this word's `y_3` (the periodic offsets `B_{kn}` converge geometrically to the same algebraic fixed point as `k вҶ’ вҲһ`, the periodic case of Theorem `14.15.3.3`'s limit). If integrally realized by an actual periodic orbit, `14.14.8.4`'s reconciliation already identifies that fixed point with the classical rational cycle candidate of cycles.md `В§12.1` вҖ” not a lever, and not re-derived here.
- **The general locus.** Characterizing which bi-infinite words are integrally realized вҖ” equivalently, which `(y_2,y_3) вҲҲ Z_2 Г— Z_3` lie on the diagonal вҖ” **is the Bridge in symbolic form** (bridge.md `В§16`): closing it for typical words is the equidistribution question (aeh.md `В§13`, this section's `14.15.2` pointer), closing it for periodic words beyond the classical candidate is the cycle-rigidity question (`16.4.4`). The formulation above is the deliverable; per the brief, no attempt is made here to characterize the locus beyond this definition, the trivial direction, and the periodic reconciliation.

## 14.15.4. The unique-predecessor lemma, the finite bicylinder corollary, and the positive realization height

*(The letter-prescribed predecessor of a positive door is automatically positive, and nested-window monotonicity holds only around a *fixed* origin, not a sliding one. Item (b) closes a small question (no finite two-sided obstruction, ever); item (c) is a *formulation* вҖ” the diagonal compatibility locus of `14.15.3`(c) renamed as a boundedness question at the archimedean place вҖ” and moves nothing about the Bridge.)*

### (a) The unique-predecessor lemma

**Theorem 14.15.4.1 (unique-predecessor lemma).** Fix a live door `z` and a letter `(m,r)`, `m,r вүҘ 1`. There is at most one odd integer `y` with `G(y) = z` and `stratum(y) = (m,r)`, namely

```text
y = 2^m (2^r z + 1) / 3^m вҲ’ 1,
```

which is an odd integer iff `3^m | 2^r z + 1`, and when it is, `y` is automatically positive and lies on stratum exactly `(m,r)` вҖ” both `v_2` conditions are forced by the construction, with no separate check.

**Proof.** *Uniqueness.* Suppose `y` satisfies `G(y)=z` and `stratum(y)=(m,r)`. Write `q := (y+1)/2^m`, an odd integer since `v_2(y+1)=m`. By Definition `14.14.3.1` (reverse.md), `G(y) = (3^m q вҲ’ 1)/2^{v_2(3^m q вҲ’ 1)}`, and `v_2(3^m q вҲ’ 1) = r` by hypothesis, so `G(y)=z` reads `3^m q вҲ’ 1 = 2^r z`, i.e. `q = (2^r z + 1)/3^m` вҖ” one equation, at most one solution `q`, hence at most one `y = 2^m q вҲ’ 1`.

*Existence and the forced conditions.* Suppose `3^m | 2^r z + 1`; set `q := (2^r z+1)/3^m`, `y := 2^m q вҲ’ 1`. Since `z` is odd and `r вүҘ 1`, `2^r z + 1` is odd, and dividing the odd `3^m` into it preserves oddness: `q` is odd. So `y+1 = 2^m q` with `q` odd, giving `v_2(y+1) = m` exactly вҖ” the prescribed `m` is forced, not assumed. And `3^m q вҲ’ 1 = 2^r z` by construction, `z` odd, so `v_2(3^m q вҲ’ 1) = v_2(2^r z) = r` exactly вҖ” the prescribed `r` is likewise forced. Hence `stratum(y) = (m,r)` exactly and `G(y) = (3^m q вҲ’ 1)/2^r = z`. Positivity: `q = (2^r z+1)/3^m` is a quotient of two positive numbers that is, by hypothesis, an integer, hence a positive integer (never zero, since `2^r z + 1 вүҘ 3`), so `q вүҘ 1` and `y = 2^m q вҲ’ 1 вүҘ 2^m вҲ’ 1 вүҘ 1` (`m вүҘ 1`). вҲҺ

**Content.** This is injectivity of `G` on a fixed stratum, made explicit and inverted: `14.14.4.1` already exhibits `G`, restricted to a stratum, as affine with nonzero rational slope `3^m/2^{m+r}`; the displayed formula is exactly that affine map inverted, and the nonzero slope is exactly why a solution, when it exists, is unique. The external suggestion packaged into this brief named "negative ... deep predecessors" as a failure mode; the algebra above shows this cannot happen вҖ” the two real failure modes are exactly the admissibility congruence `3^m | 2^r z + 1` failing, and (a separate matter, addressed in `14.15.4`(b)вҖ“(c)) liveness of a chain's deepest door.

**Reconciliation with the backward branching law (`14.2.4`) вҖ” cited, not re-derived; no mismatch found.** Set `N := 2^r z + 1`, `D := v_3(N)`. Substituting into `14.1.1`'s representative-door formula `y_a = 2^{DвҲ’a} 3^a О© вҲ’ 1` (`О© = N/3^D`) at index `a = D вҲ’ m` gives `3^{DвҲ’m}О© = N/3^m = q`, hence `y_a = 2^m q вҲ’ 1` вҖ” exactly the formula above. So the letter-prescribed predecessor of this theorem is `14.1.1`'s own representative door of the predecessor state `(О©, D)`, at the representative index `a = D вҲ’ m` that `m` names implicitly and exit valuation `s = r`; and `14.2.4`'s law `D = 1 + v_3(r вҲ’ M_3(z))` computes, via the anchor, the same depth `D = v_3(2^r z + 1)` this construction needs directly. The two routes agree exactly.

### (b) The finite bicylinder corollary

**Theorem 14.15.4.2 (finite bicylinder corollary).** Let `V = ((m_{вҲ’p},r_{вҲ’p}),вҖҰ,(m_{вҲ’1},r_{вҲ’1}))` and `U = ((m_0,r_0),вҖҰ,(m_{qвҲ’1},r_{qвҲ’1}))` be finite words (past of length `p`, future of length `q`). There is a positive live door `y_0` with an actual chain of live-door predecessors `y_{вҲ’1},вҖҰ,y_{вҲ’p}` realizing `V` exactly (`G(y_{вҲ’i})=y_{вҲ’i+1}`, `stratum(y_{вҲ’i})=(m_{вҲ’i},r_{вҲ’i})` for `i=1,вҖҰ,p`, deepest door `y_{вҲ’p}` live, intermediate doors live automatically) and whose forward orbit realizes `U` exactly (`stratum(G^j(y_0))=(m_j,r_j)` for `j=0,вҖҰ,qвҲ’1`).

**Proof.** Concatenate `V` and `U` into the length-`(p+q)` word `W`. By the finite-itinerary cylinder theorem (`14.15.1.5`), the followers of `W` form one residue class, nonempty and infinite (`14.15.1.6`); by `14.15.1.7`, this class contains infinitely many live doors. Pick one, `y_{вҲ’p}`, and set `y_{вҲ’p+i} := G^i(y_{вҲ’p})` for `i=1,вҖҰ,p+q`, `y_0 := G^p(y_{вҲ’p})`. By the definition of "follows `W`" (`14.15.1.2`), `stratum(y_{вҲ’p+i})` is `W`'s `i`-th letter for `i=0,вҖҰ,p+qвҲ’1`: the first `p` letters, realized starting at `y_{вҲ’p}`, are `V`; the remaining `q`, realized starting at `y_0`, are `U`. `y_{вҲ’p}` is live by choice; every later door in the chain is a `G`-image, hence live automatically (`14.14.3.2`(3) / `14.14.7.1`'s totality-and-live-image fact, which needs no liveness hypothesis on its input). Each step of the resulting chain is, by `14.15.4.1`'s uniqueness, exactly the letter-prescribed predecessor of the next door on the prescribed letter вҖ” not a separate construction, the same one. вҲҺ

**Consequence, stated flatly, in `14.15.2`'s register.** Every finite two-sided itinerary window is realized by a positive live integer segment вҖ” there is no finite two-sided obstruction. This extends `14.15.2`'s verdict (the word is free at every finite *forward* depth) to finite windows with prescribed past: prescribing a past costs nothing either, at any finite depth.

### (c) The positive realization height

**Definition 14.15.4.3 (window, positive realization height).** Fix a bi-infinite word `W = (вҖҰ,(m_{вҲ’1},r_{вҲ’1}),(m_0,r_0),(m_1,r_1),вҖҰ)` and a window `(p,q)`, `p,q вүҘ 0`, around `W`'s *fixed* origin (position `0`, held fixed throughout вҖ” windows are never re-centred). Let

```text
R_{p,q}(W) := { y_0 :  y_0 a positive live door,
                stratum(G^j(y_0)) = (m_j,r_j) for j = 0,вҖҰ,qвҲ’1,
                and the 14.15.4.1 letter-prescribed backward chain
                (predecessor on (m_{вҲ’1},r_{вҲ’1}), its predecessor on
                (m_{вҲ’2},r_{вҲ’2}), вҖҰ) exists to depth p with live
                deepest door }.
```

`H_{p,q}(W) := min R_{p,q}(W)` whenever `R_{p,q}(W) вү  вҲ…`.

**Theorem 14.15.4.4 (basic properties).**

1. `R_{p,q}(W) вү  вҲ…` for every `p,q вүҘ 0`, so `H_{p,q}(W)` is always defined.
2. `R_{p,q+1}(W) вҠҶ R_{p,q}(W)` and `R_{p+1,q}(W) вҠҶ R_{p,q}(W)`, hence `R_{p',q'}(W) вҠҶ R_{p,q}(W)` whenever `p' вүҘ p, q' вүҘ q` вҖ” in particular `R_{p+1,q+1}(W) вҠҶ R_{p,q}(W)` (fixed origin).
3. `H_{p,q}(W)` is a positive integer, nondecreasing in `(p,q)` under the product order.

**Proof.** (1) is `14.15.4.2` applied to `W`'s own length-`p` past and length-`q` future around the origin. (2): if `y_0 вҲҲ R_{p,q+1}(W)`, its forward orbit realizes letters `0,вҖҰ,q`, in particular the shorter prefix `0,вҖҰ,qвҲ’1`, and its depth-`p` backward condition is untouched вҖ” `y_0 вҲҲ R_{p,q}(W)`. If `y_0 вҲҲ R_{p+1,q}(W)`, its depth-`(p+1)` chain `y_{вҲ’1},вҖҰ,y_{вҲ’(p+1)}` restricts to the depth-`p` prefix `y_{вҲ’1},вҖҰ,y_{вҲ’p}`, whose deepest door `y_{вҲ’p} = G(y_{вҲ’(p+1)})` is live automatically (a `G`-image of the live `y_{вҲ’(p+1)}`) вҖ” `y_0 вҲҲ R_{p,q}(W)`; the forward condition is untouched. Chaining the two gives the general case. (3): `R_{p,q}(W)` is a nonempty set of positive integers, so its minimum exists; (2)'s inclusions give (subset's minimum) `вүҘ` (superset's minimum), i.e. `H_{p',q'}(W) вүҘ H_{p,q}(W)` whenever `p'вүҘp, q'вүҘq`. вҲҺ

**Theorem 14.15.4.5 (equivalence theorem).** `W` is integrally realized (`14.15.3.5`) if and only if `(H_{p,q}(W))_{p,q}` is bounded, if and only if it is eventually constant.

**Proof.** *Bounded вҹә eventually constant.* Eventually constant trivially implies bounded. Conversely suppose `H_{p,q}(W) вүӨ B` for all `p,q`. Along the diagonal, `(H_{n,n}(W))_n` is a nondecreasing sequence of positive integers bounded by `B`, hence eventually constant, say `= y_0` for `n вүҘ N`. For `p,q вүҘ N`: `H_{p,q}(W) вүҘ H_{N,N}(W) = y_0` (monotonicity, `(p,q)вүҘ(N,N)`) and `H_{p,q}(W) вүӨ H_{n,n}(W) = y_0` with `n=max(p,q)вүҘN` (monotonicity, `(n,n)вүҘ(p,q)`), so `H_{p,q}(W)=y_0` for *all* `p,qвүҘN` вҖ” eventually constant as a full net, not merely along the diagonal.

*Integrally realized вҹ№ bounded.* Let `y_0^*` realize `W` (`14.15.3.5`): its forward orbit realizes the entire right-infinite half, and it has an actual integer chain of live-door predecessors realizing the entire left-infinite half. For any `p,q`, the depth-`p` prefix of that actual chain is a chain of live-door predecessors realizing the first `p` past letters; by `14.15.4.1`'s uniqueness, any predecessor satisfying `G(y)` equal to the next door and carrying the prescribed stratum equals the unique formula value, so this actual chain coincides, step for step, with the letter-prescribed one. Hence `y_0^* вҲҲ R_{p,q}(W)` for every `p,q`, giving `H_{p,q}(W) вүӨ y_0^*` uniformly.

*Bounded вҹ№ integrally realized.* By the first paragraph, `H_{p,q}(W) = y_0` for all `p,q вүҘ N`. Fix any `(p,q)`; taking `p',q' вүҘ max(p,q,N)`, `(2)` gives `R_{p',q'}(W) вҠҶ R_{p,q}(W)`, and `y_0 вҲҲ R_{p',q'}(W)` (the minimum of a nonempty set is a member of it), so `y_0 вҲҲ R_{p,q}(W)`. As `(p,q)` was arbitrary, `y_0 вҲҲ вӢӮ_{p,q} R_{p,q}(W)`. Letting `qвҶ’вҲһ` in the forward condition: `y_0`'s entire forward orbit realizes `W`'s right-infinite half, deterministically. For the backward half: `y_0` has, for every `p`, a letter-prescribed chain to depth `p` with live deepest door; by `14.15.4.1`, the letter-prescribed predecessor at each step is *unique*, so the depth-`(p+1)` chain's nearest `p` entries coincide exactly with the depth-`p` chain вҖ” the chains are **compatible under truncation because the prescribed predecessor is unique**, and they union into a single infinite chain `y_{вҲ’1},y_{вҲ’2},вҖҰ` (`G(y_{вҲ’(i+1)})=y_{вҲ’i}` for every `iвүҘ0`) realizing the entire left-infinite half. Liveness: `y_0` is live by definition of `R` (any `R_{p,q}(W)` with `p,qвүҘ0` requires it); each `y_{вҲ’i}` (`iвүҘ1`) is live because it is a `G`-image of the next-deeper door of any chain of depth `>i` вҖ” automatic regardless of whether that deeper door itself needed an explicit check (`14.14.3.2`(3)). So `y_0`, with this chain, witnesses that `W` is integrally realized. вҲҺ

**Remark (the three-notion separation, stated once).** Three notions of "does a finite window occur," strictly increasing in strength. **Finite symbolic legality** вҖ” always: the itinerary language is the full shift (`14.15.2`), every finite word a legal transition sequence, nothing to fail. **Finite positive realization** вҖ” always: `14.15.4.2`, every finite two-sided window is realized by an actual positive live integer segment. **Realization at all depths simultaneously** вҖ” a single integer realizing the *entire* bi-infinite word вҖ” is strictly stronger, and by `14.15.4.5` is exactly equivalent to boundedness of the positive realization height. Its failure mode, when it fails, is `H_{p,q}(W) вҶ’ вҲһ`: the minimal witnesses guaranteed to exist at every finite depth by the second notion are forced to escape to infinity as the window grows, so no single integer serves every window at once. This is the precise sense вҖ” **archimedean**, at the place the `Z_2 Г— Z_3` diagonal embedding forgets вҖ” in which a word can be finitely compatible everywhere and yet realized by no integer.

**Verified** вҖ” `experiments/realization_height.py`, functions `test_trivial_word_height`, `test_monotonicity`, on exactly the two sanity families the brief permits (no growth study of `H` performed or attempted). Trivial all-`(1,1)` word: `y_0=1` (`stratum(1)=(1,1)`, `G(1)=1`, the trivial fixed point) checked to be a member of `R_{n,n}` for `n=0,вҖҰ,4` directly вҖ” since `1` is the least positive odd integer, membership alone proves `H_{n,n}=1` exactly, no search needed вҖ” `0` failures at any `n`. **Framing (read before the theorems below).** Proving the converse **completes the formulation** of `14.15.3`(c); it does not open the locus. The combined statement вҖ” a bi-infinite word is integrally realized iff `yвӮӮ(W) = yвӮғ(W) =` a single positive odd integer вҖ” is an exact characterization of the diagonal locus **in terms of the word's own two adelic limits**, and deciding whether those limits coincide at a positive integer, for any given family of words, is exactly as hard as it was: it *is* the Bridge (bridge.md В§16), under a sharper name. ### (a) The admissibility-class lemma

**Setup and the order convention.** Write a finite left word two ways, matching two contexts already on file. `V = ((m_{вҲ’1},r_{вҲ’1}),вҖҰ,(m_{вҲ’n},r_{вҲ’n}))` вҖ” shallowest letter first, `m_{вҲ’1}` nearest the present door вҖ” is the natural indexing for "a window of the past" (matching `14.15.3`(b)'s own bi-infinite word notation). `14.14.8.2`'s (reverse.md) composed-affine constants `A_n, B_n`, however, are defined for a word processed **in application order** вҖ” the letter applied first (to the innermost, deepest door) listed first вҖ” which for a backward chain read forward is `V`'s **reverse**, `((m_{вҲ’n},r_{вҲ’n}),вҖҰ,(m_{вҲ’1},r_{вҲ’1}))`, exactly `14.15.3`(b)'s own `W_{(-n:0)}`. Below, `A_n, B_n` always mean `14.14.8.2` applied to this deepest-first ordering, and `M_n := ОЈбөў m_{вҲ’i}` (one quantity either way, since it is just a sum).

**Theorem 14.15.5.1 (admissibility-class lemma).** For a finite left word `V = ((m_{вҲ’1},r_{вҲ’1}),вҖҰ,(m_{вҲ’n},r_{вҲ’n}))`, `M_n = ОЈбөў m_{вҲ’i}`: among positive odd integers, the `z` admitting the letter-prescribed backward chain to depth `n` (each step existing per `14.15.4.1`'s criterion, applied successively) form **exactly one residue class mod `3^{M_n}`, namely `B_n mod 3^{M_n}`**, `A_n, B_n` the composed-affine constants of `V` read deepest-first, as above.

**Remark (liveness is not assumed at intermediate doors вҖ” review addition, 2026-07-16).** `14.15.4.1` is stated for a *live* door `z`, but its proof nowhere uses `3 вҲӨ z`: the existence criterion, the predecessor formula, and the uniqueness argument hold verbatim for every odd `z` вҖ” the same generality `14.15.1.1`'s remark records for `stratum` and `G`. The induction below applies the criterion at intermediate chain doors regardless of their liveness (a finite prefix's deepest door may be dead); the chain semantics here does not require liveness, and it enters only at `14.15.5.3`'s infinite union, where every door is a `G`-image and liveness is automatic.

**Proof.** Induction on `n`. *Base* (`n=1`): by `14.15.4.1`, `z` admits a predecessor on letter `(m_{вҲ’1},r_{вҲ’1})` iff `3^{m_{вҲ’1}} | 2^{r_{вҲ’1}}z+1`, iff `z вүЎ -2^{-r_{вҲ’1}} (mod 3^{m_{вҲ’1}})` (`2` a unit mod every power of `3`) вҖ” one class. `B_1 = ОІ_{(m_{вҲ’1},r_{вҲ’1})}` (the length-one word's own composed constant), and `ОІ = (3^m-2^m)2^{-(m+r)} = 3^m2^{-(m+r)} - 2^{-r} вүЎ -2^{-r} (mod 3^m)` directly (the first term vanishes mod `3^m`, being `3^m` times a `3`-adic unit). So the base class is exactly `B_1 mod 3^{m_{вҲ’1}}`.

## 14.15.6. The signed diagonal: `14.15.1`вҖ“`14.15.5` over the nonzero odd integers

### (a) The signed domain, the singular point, and sign preservation

**Definition 14.15.6.1 (signed door space).** The **signed door space** is the set of nonzero odd integers `y вү  вҲ’1`. As before, `y` is **live** iff `3 вҲӨ y`. `stratum(y) = (m,r)` and `G(y)` are computed by `14.15.1.1`'s formulas verbatim (`m = v_2(y+1)`, `q=(y+1)/2^m`, `r=v_2(3^mq-1)`, `G(y)=(3^mq-1)/2^r`) вҖ” algebra that nowhere references the sign of `y`.

**Theorem 14.15.6.2 (totality and sign preservation).** For every `y` in the signed door space:

1. `stratum(y)` and `G(y)` are defined (`m,r` both finite).
2. `3 вҲӨ G(y)` (`G`'s image is always live).
3. `G(y)` has the **same sign as `y`**, without exception вҖ” `G(y) > 0` whenever `y>0`, and `G(y) < 0` whenever `y<0`.
4. **The one genuine sector asymmetry.** `G(y) = вҲ’1` is possible only when `y<0` вҖ” e.g. `G(вҲ’3) = вҲ’1` exactly вҖ” never when `y>0`. (3) is new algebra, the one place positivity entered `14.15.1`вҖ“`14.15.5` structurally rather than as a hypothesis label: if `y>0` then `q=(y+1)/2^m>0` is a positive odd integer, so `3^mq вүҘ 3` and `val = 3^mq-1 вүҘ 2 > 0`, giving `G(y)=val/2^r>0`. ### (a) The fixed-point arithmetic, the rotation lemma, and the adelic anchor

**Lemma 14.15.9.1 (fixed-point arithmetic).** For every period `P`, with `N := ОЈ_{i=0}^{pвҲ’1} (3^{m_i}вҲ’2^{m_i})В·3^{ОЈ_{j>i}m_j}В·2^{ОЈ_{j<i}(m_j+r_j)}` and `D := 2^{S_P}вҲ’3^{M_P}`:

**Proof.** (1) Unwinding `14.14.8.2`'s recursion, `B_P = ОЈ_i ОІ_iВ·О _{j>i}Оұ_j`, and multiplying through by `2^{S_P}` clears every denominator: `2^{S_P}B_P = N` term by term. Parity: the `i = 0` term is oddВ·oddВ·`2^0`, odd; every `i вүҘ 1` term carries `2^{ОЈ_{j<i}(m_j+r_j)}` with exponent `вүҘ 2`, even вҖ” so `N` is odd. Mod `3`: every term with `i < pвҲ’1` carries `3^{ОЈ_{j>i}m_j}` with exponent `вүҘ 1`, `вүЎ 0`; the `i = pвҲ’1` term is `вүЎ вҲ’2^{m_{pвҲ’1}+ОЈ_{j<pвҲ’1}(m_j+r_j)} вүў 0` вҖ” so `3 вҲӨ N`. **Lemma 14.15.9.2 (rotation lemma).** Let `F_i` be the composed affine map of `P^{(i)}` (letters `i, i+1, вҖҰ, i+pвҲ’1` in application order), with unique fixed point `y^*_i` (`A_P вү  1`: `vвӮӮ(A_P) = вҲ’S_P вү  0`). Then:

1. **(One affine orbit.)** `y^*_{i+1} = Оұ_i y^*_i + ОІ_i` for every `i` (indices mod `p`): the `p` rotated fixed points are a single orbit of the per-letter affine maps, closing up after one period.
2. **(One denominator.)** All `p` rotations have the **same** reduced denominator `q`.
3. **(The mod-3 residues.)** `Оө_i := y^*_i mod 3 = a_iВ·q^{вҲ’1} mod 3 вҲҲ {1,2}` (well defined by Lemma `14.15.9.1` applied to `P^{(i)}`), and `Оө_{i+1} = вҲ’(вҲ’1)^{r_i} mod 3` вҖ” `1` if `r_i` is odd, `2` if `r_i` is even: each rotation's residue is pinned by the *preceding* letter's `r`-parity alone.

**Proof.** (1) Since `g_{i+p} = g_i`, both `F_{i+1}вҲҳg_i` and `g_iвҲҳF_i` equal `g_iвҲҳg_{i+pвҲ’1}вҲҳвҖҰвҲҳg_{i+1}вҲҳg_i` read with different bracketing: `F_{i+1}вҲҳg_i = g_iвҲҳF_i`. Forward: writing `y^*_i = a_i/q_i` reduced, `g_i(y^*_i) = (3^{m_i}a_i + (3^{m_i}вҲ’2^{m_i})q_i)/(2^{m_i+r_i}q_i)`, so the reduced denominator `q_{i+1}` divides `2^{m_i+r_i}q_i`; being odd (Lemma `14.15.9.1` for `P^{(i+1)}`), no factor of `2` can survive reduction into it, so `q_{i+1} | q_i`. Backward: `g_i^{вҲ’1}(v) = (2^{m_i+r_i}v вҲ’ (3^{m_i}вҲ’2^{m_i}))/3^{m_i}` likewise gives `q_i | 3^{m_i}q_{i+1}`, and `q_i` is coprime to `3`, so `q_i | q_{i+1}`. Hence `q_{i+1} = q_i` for every `i`. (3) Mod `3` (in `ZвӮғ`, where every `y^*_i` lives): `vвӮғ(Оұ_i) = m_i вүҘ 1`, so `y^*_{i+1} = Оұ_i y^*_i + ОІ_i вүЎ ОІ_i = (3^{m_i}вҲ’2^{m_i})В·2^{вҲ’(m_i+r_i)} вүЎ вҲ’2^{вҲ’r_i} вүЎ вҲ’(вҲ’1)^{r_i} (mod 3)` (`2 вүЎ вҲ’1`). вҲҺ
