Anchor digit visualizer

First-tier visualizations for the single-sequence digit-structure search (anchors.md §17.9): does one M(ω) bit string, read on its own, show self-similarity, periodicity, or bias — as opposed to the cross-sectional (many-orbit) statistics aeh.md already checked. Companion to the statistical battery in experiments/anchor_digit_structure.py and experiments/anchor_compressibility.py (brief: briefs/anchor-digit-search-brief.md).

Pick an ω from the bottom group (small, historically flagged in archive/appendix-a.md A.4.6 — expect visible structure, e.g. ω=1 and ω=3 are exact algebraic degeneracies) or the bulk group (large, generic ω — the actually-open regime) and compare. Bit strings are 8192 bits, computed via the same fresh 2-adic-log generator as the statistical battery (experiments/anchor_visualize.py), validated against the worked-example table in anchors.md §17.7.

Recurrence plot

Compares a w-bit window at every sampled offset i against every offset j; darker = more similar (lower Hamming distance). A random string lights up only the diagonal, in a faint uniform haze off it. Self-similarity or periodicity shows as off-diagonal stripes or blocks.

Chaos game representation

Walks a point halfway toward a target corner of a square, one target per step. True randomness fills the plotted region as uniform fog; bias or forbidden patterns show as visible gaps or self-similar (fractal) structure.
