# 14. The Reverse Dynamics: a 3-adic Mirror

## 14.1. Predecessor structure


**Proof.** `F(ŌČ,d) = (ő©,D)` iff `x_exit(ŌČ,d) = (3^d ŌČ ‚ąí 1)/2^s` equals some representative `y`, i.e. `3^d ŌČ = 2^s y + 1` with `s = v‚āā(3^d ŌČ ‚ąí 1)` consistent. ## 14.7. Digit-determinacy: the 3-adic mirror

Paper Theorem 3.5 (`thm:deltaM`) is proved by chaining three digit-determinacy facts (a)‚Äď(c) about `N(u)`, `C`, `ŌČ_next`. Their mirrors, in the notation of 14.1‚Äď14.2 (door `y`, branch `s`, `N = 2^s y + 1`, predecessor `(ŌČ,d)`, `d = v‚āÉ(N)`, `ŌČ = N/3^d`):

**Proposition 14.7.1 (mirror digit-determinacy facts).**

```text
(a')  M‚āÉ(y) mod 3^k        is determined by  y mod 3^(k+1),
(b')  N     mod 3^q        is determined by  y mod 3^q  and  s mod 3^(q‚ąí1)  (parity of s fixed by y mod 3),
(c')  ŌČ = N/3^d mod 3^r    is determined by  N mod 3^(d+r)  and  d  (exact division).
```

**Proof.** (a') `2` is a primitive root mod `3^(k+1)` (order `2¬∑3^k`), so `2^t mod 3^(k+1)` determines `t mod 2¬∑3^k`, i.e. both `t`'s parity and its `Z‚āÉ`-truncation mod `3^k`; `y mod 3^(k+1)` determines `‚ąí1/y mod 3^(k+1)`, hence `t = M‚āÉ(y) mod 3^k`. (b') restricted to the fixed-parity coset of exponents, `2`'s effective order mod `3^q` is `3^(q‚ąí1)`, so `2^s mod 3^q` is determined by `s mod 3^(q‚ąí1)`; `y` enters `N` linearly. (c') dividing `N` by `3^d` to get a residue mod `3^r` is exact division *within the same prime* ‚ÄĒ no analogue of a generator-order fact is needed. ‚ąé

**Theorem 14.7.2 (mirror of Thm 3.5).** For every window depth `W` and target `r`: given `y mod 3^(W+1)` and `s mod 3^W`, if `őĶ := (s ‚ąí M‚āÉ(y)) mod 3^W` is nonzero, then `d = 1 + v‚āÉ(őĶ)` is exact; and if `W ‚Č• d + r`, the *same* truncations of `y, s` (via (a')‚Äď(c'), chained exactly as Thm 3.5 chains (a)‚Äď(c)) determine `ŌČ mod 3^r`.

**Finding ‚ÄĒ the missing cross-prime step (this *is* the honest mirror, not a gap).** Forward's fact (c) needs the order of `3` mod `2^r` because `ŌČ_next` is obtained by dividing a *2-adically analyzed* quantity `C` by a `3`-power ‚ÄĒ a genuine cross-prime removal. **Finding ‚ÄĒ trichotomy collapses to dichotomy.** Forward's valuation law genuinely splits: six of eight residue classes fix `s ‚ąą {1,2}` as a class constant at *zero* window cost, and only the two lifting classes need the window ‚ÄĒ three runtime outcomes (non-lifting-decided / lifting-decided / lifting-undecided). The mirror trichotomy genuinely degenerates to a dichotomy ‚ÄĒ decided/undecided, no free third branch ‚ÄĒ because `(Z/3^q)^*` is cyclic (generated by `2`, no split), unlike `(Z/2^q)^*` which has the extra `Z/2` factor that gives forward's six non-lifting classes their zero-cost constants.

**Verification.** `K ‚ąą {2,4,6,8}`, ‚Čą13,200 trials each: zero decision errors at every `K`; undecided rate close to `3^(‚ąíK)` (`0.108` vs `0.111` at `K=2`; `0.00083` vs `0.00137` at `K=6`; `0.00008` vs `0.00015` at `K=8`); zero deep-bound violations (every "undecided" case has true `d > K`). ## 14.10. The dual ladder: predecessors at adjacent branches, fixed door

Ladder.md ¬ß15 relates `(ŌČ,d)` and `(ŌČ,d+1)` ‚ÄĒ same core, adjacent depth. The mirror fixes a door `y` and relates predecessors at adjacent branches `s` and `s+2` (the forced step: `s` is confined to one parity by 14.1.1, so `s+1` is not a valid branch).

**Theorem 14.10.1 (ladder dichotomy).** Write `N(y,s) = 2^s y + 1 = 3^d ŌČ`. Then

```text
N(y, s+2) = 4¬∑N(y,s) ‚ąí 3,
```

and this forces:

```text
d(y,s) = 1   ‚üĻ   ŌČ(y,s+2) = T‚āÉ(ŌČ(y,s)),   d(y,s+2) = 1 + v‚āÉ(4ŌČ(y,s) ‚ąí 1),   where T‚āÉ(ŌČ) := (4ŌČ‚ąí1)/3^(v‚āÉ(4ŌČ‚ąí1)),
d(y,s) ‚Č• 2   ‚üĻ   ŌČ(y,s+2) = 4¬∑3^(d‚ąí1)¬∑ŌČ(y,s) ‚ąí 1   exactly,   and   d(y,s+2) = 1   exactly.
```

**Proof.** `N(y,s+2) = 2^(s+2)y + 1 = 4(2^s y) + 1 = 4(N(y,s) ‚ąí 1) + 1 = 4N(y,s) ‚ąí 3`. Write `N = 3^d ŌČ`; `v‚āÉ(4N) = d` (`4` a `3`-adic unit), `v‚āÉ(3) = 1`. If `d ‚Č† 1`: no valuation collision, so `v‚āÉ(N') = min(d,1) = 1` by the ultrametric inequality, forcing `N'/3` coprime to `3`; and `N'/3 = 4N/3 ‚ąí 1 = 4¬∑3^(d‚ąí1)ŌČ ‚ąí 1` exactly, giving the `d ‚Č• 2` branch (this also covers `d=0`, impossible for a valid state). If `d = 1`: `N = 3ŌČ`, so `N' = 3(4ŌČ ‚ąí 1)`, a genuine collision ‚ÄĒ `v‚āÉ(N') = 1 + v‚āÉ(4ŌČ‚ąí1)` and `ŌČ(y,s+2) = (4ŌČ‚ąí1)/3^(v‚āÉ(4ŌČ‚ąí1))`, the `T‚āÉ` branch. ‚ąé

**Finding ‚ÄĒ the gate, and the forced step.** The pivot `d=1` vs `d‚Č•2` is exactly Theorem 14.2.4's own first digit: `d=1` iff `v‚āÉ(s ‚ąí M‚āÉ(y)) = 0`. As the brief anticipated, the dual ladder's tear-line is gated by the 3-adic anchor, exactly mirroring the forward ladder's tear-line being gated by `s(ŌČ,d)=1` via the 2-adic anchor `M(ŌČ)`. The coefficient `4` (not forward's `3`) is not a broken mirror: it is `2^2`, forced by the step size `2` (not `1`) that the parity condition of 14.1.1 imposes on `s` ‚ÄĒ the ladder's "unit step" is a lattice-of-index-2 step here, a direct and expected consequence of already-proved structure, not a new asymmetry.

## 14.11. Scope and standing

Backward reachability of every valid state from `(1,1)` is *identical* to the Collatz conjecture (Theorem 9.8.3) ‚ÄĒ this front offers no discount on the hard part, and per the digit-budget principle its unbounded-depth content is the same as the forward front's (3-adic digits now, rather than 2-adic).
